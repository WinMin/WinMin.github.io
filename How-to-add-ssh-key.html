CTF-Rank运维日记 （二）

Swing
2017-08-16
环境配置
ssh
第一步：

客户机 生成公钥
ssh-keygen -t rsa

第二步：

将公钥传到服务器
scp id_rsa.pub root@ip:/root/.ssh/
将公钥导入keys文件
cat /root/.ssh/id_rsa.pub >> /root/.ssh/authorized_keys

第三步：

修改ssh配置文件
Vim /etc/ssh/sshd_config

第四步：

重启服务
service sshd restart
